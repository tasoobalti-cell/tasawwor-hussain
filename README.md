# tasawwor-hussain

A polished personal portfolio website for Tasawwar Hussain.

## Project structure

- `index.html` — main portfolio page
- `styles.css` — page styling and animations
- `script.js` — scroll reveal animation support
- `.gitignore` — ignored files for GitHub

## Development

Open `index.html` in a browser to preview the site locally.

## GitHub Pages

This repository can be published with GitHub Pages from the `main` branch. Once the repository is pushed, enable Pages in the repository settings and select `main` as the source.

## Notes

The design uses a dark gradient theme with animated section reveals and polished project/skills cards. The same color palette is preserved, and the layout is ready for GitHub deployment.
